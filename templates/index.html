How to Use the Website Change Monitor

  1. Enter a website URL and check interval (in minutes) in the form below.
  2. Click "Add Website" to start monitoring the website.
  3. The website will appear in the list with its current status.
  4. Click on a website URL to visit it (this will also update its "last visited" time).
  5. Use the "Update Interval" button to change the check frequency.
  6. Use the "Remove" button to stop monitoring a website.

Status Legend:

Changes were made since the last time you visited
No changes since your last visit, or you've already seen the changes
The website is unreachable or check is overdue
